# Security Policy

## Reporting a Vulnerability

If you find a security issue in pdfdb, please report it privately rather than
opening a public issue.

- Email: loai.misto@gmail.com
- Or open a private security advisory:
  https://github.com/lmist/pdfdb/security/advisories/new

Please include enough detail for the issue to be reproduced (affected version,
steps, expected vs actual behavior). You'll typically get an acknowledgment
within a few days.
